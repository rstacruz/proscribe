    Project

    A project.

    Internal usage

    p = Project.new(config_file)

    Attributes

    • config Returns a hash of the project's configuration, pulled from Scribefile.
    • dir Returns the path to the temporary Proton project.
    • manual_path Returns the absolute path to the projects's manual.
    • root Returns the root path.

    Methods

    • make Creates the temp dir from the project's manual and inline comments.
