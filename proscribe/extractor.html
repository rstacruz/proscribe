    Extractor

    Extracts comments from list of files.

    Description

    Gets the ones with comment blocks starting with [...]

    Common usage

    ex = Extractor.new(Dir['./**/*.rb'])
ex.blocks

ex.blocks.map! { |b| b.file = "file: #file (b)" }

ex.write!('manual/')       # Writes to manual/
